Okhttp 源码分析

发表于 2021-10-10 更新于 2023-03-05 分类于 Third

OkHttp 是 Square 公司开源的网络请求框架。有如下优点：

  • 支持 HTTPS；
  • 支持 HTTP2/SPDY；
  • Socket 自动选择最好路线，并支持自动重连；
  • 连接池（ConnectionPool）减少了请求延迟；
  • 透明 GZip 压缩；
  • HTTP 缓存，只支持 GET 请求；
  • 当网络出现问题时，OkHttp 会自动重试一个主机的多个 IP 地址；
  • 拦截器机制，轻松处理请求与响应。
